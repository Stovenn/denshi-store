## Description

An electronic shop inventory API developed with [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ npm install
$ docker build -t denshi-store .
```

## Running the app

```bash
$ docker compose up
```

## Test

```bash
# unit tests
$ npm run test

```
